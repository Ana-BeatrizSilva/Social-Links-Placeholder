# Português

## Sobre o Projeto

**Social Links Placeholder** é uma interface estática desenvolvida com HTML5 e CSS3, criada com o objetivo de praticar conceitos fundamentais de desenvolvimento front-end, incluindo estruturação semântica, estilização de componentes, organização visual e construção de layouts modernos.

O projeto simula um cartão de perfil contendo informações básicas de um desenvolvedor fictício e uma coleção de links para redes sociais. Embora apresente uma estrutura semelhante à de uma página de portfólio pessoal, seu conteúdo possui caráter ilustrativo e foi desenvolvido exclusivamente para fins de aprendizado e prática.

Durante o desenvolvimento foram aplicados conceitos importantes de design de interface, tipografia, alinhamento, responsividade, efeitos visuais, hierarquia de informações e boas práticas de organização de código.

Este projeto foi desenvolvido com base em um modelo disponibilizado pelo Frontend Mentor, sendo adaptado e personalizado para fins educacionais.

---

# English

## About the Project

**Social Links Placeholder** is a static interface built with HTML5 and CSS3, created to practice fundamental front-end development concepts, including semantic structure, component styling, visual organization, and modern layout design.

The project simulates a profile card containing basic information about a fictional developer and a collection of social media links. Although it resembles a personal portfolio page, all content is intended as placeholder material and was created exclusively for learning and practice purposes.

During development, important concepts such as interface design, typography, alignment, responsiveness, visual effects, information hierarchy, and code organization best practices were applied.

This project was developed based on a Frontend Mentor challenge template and customized for educational purposes.

---

## Learning Objectives

This project was created to practice and reinforce knowledge related to:

- HTML5 page structure;
- Semantic markup;
- Component-based content organization;
- Modern CSS3 styling techniques;
- Gradient and visual effect implementation;
- Transparency and Glassmorphism effects;
- Element alignment and centering;
- Interactive button creation;
- Hover effects and CSS transitions;
- Responsive layout development;
- Front-end development best practices.

---

## Features

- Centered profile card;
- Circular profile image;
- Basic user information display;
- Social media links section;
- Interactive buttons with hover effects;
- Modern Glassmorphism interface;
- Gradient background;
- Clean and responsive layout.
